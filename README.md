# MiniBaritone Fabric

Учебный клиентский Minecraft Fabric-мод под `1.21.11`, сделанный по промпту про упрощённый Baritone.

Это не официальный Baritone и не форк Baritone. Это отдельная минимальная реализация:
- A* поиск пути;
- команды в чате;
- движение к координатам;
- попытка обходить препятствия;
- ломание блока перед игроком;
- установка блока под путь, если включён `allowPlace`;
- поиск и добыча дерева в радиусе до 100 блоков.

Используй мод только в одиночной игре или там, где это разрешено правилами сервера.

## Команды

Основная команда:

```text
/mb
```

Длинный вариант:

```text
/minibaritone
```

### Движение к координатам

```text
/mb goto <x> <y> <z>
```

Пример:

```text
/mb goto 100 64 -200
```

### Остановить бота

```text
/mb stop
```

### Сломать блок перед собой

```text
/mb breakfront
```

### Добывать дерево

```text
/mb tree <radius>
```

Радиус ограничен от `1` до `100`.

Пример:

```text
/mb tree 100
```

### Настройки

```text
/mb settings allowBreak true
/mb settings allowBreak false
/mb settings allowPlace true
/mb settings allowPlace false
```

По умолчанию:

```text
allowBreak = true
allowPlace = true
```

## Сборка через GitHub Actions

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы из этого архива в корень репозитория.
3. Открой вкладку **Actions**.
4. Запусти workflow **Build MiniBaritone Fabric**.
5. Готовый `.jar` будет в артефактах workflow.

## Локальная сборка

Нужны:
- Java 21;
- Gradle 8.14+.

Команда:

```bash
gradle build --stacktrace
```

Готовый `.jar` появится тут:

```text
build/libs/
```

## Важное про версии

В `gradle.properties` вынесены версии Minecraft/Fabric/Loom:

```properties
minecraft_version=1.21.11
yarn_mappings=1.21.11+build.1
loader_version=0.17.2
fabric_version=0.139.1+1.21.11
loom_version=1.11-SNAPSHOT
```

Если GitHub Actions покажет, что конкретная версия Fabric API или Yarn ещё не существует / уже изменилась, поменяй только эти строки.

## Как работает

`SimplePathfinder` строит путь A* по соседним позициям:
- ходьба по ровной поверхности;
- прыжок на 1 блок;
- безопасный спуск до 3 блоков;
- избегание лавы, кактусов, огня и магмовых блоков;
- высокая стоимость ломания блоков;
- высокая стоимость установки блока.

`PathBot` исполняет путь:
- поворачивает камеру к следующей точке;
- нажимает клавиши движения через Minecraft key bindings;
- прыгает при подъёме;
- ломает блоки, если путь перекрыт и `allowBreak=true`;
- ставит блок под следующий шаг, если есть блок в хотбаре и `allowPlace=true`.

## Ограничения

Это минимальный учебный бот, поэтому он проще настоящего Baritone:
- не умеет идеально проходить паркур;
- не работает с лодками, лошадьми, элитрами;
- не строит сложные шахтёрские маршруты;
- может застревать в деревьях, воде, лианах, пещерах;
- не гарантирует обход античитов.
